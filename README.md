# TODO App

Prosta aplikacja TODO napisana w czystym JavaScript (bez frameworków).

## Funkcjonalności
- Dodawanie, edycja i usuwanie zadań (CRUD)
- Status zadania: todo / done
- Automatyczna data utworzenia
- Sortowanie po dacie utworzenia
- Walidacja danych (tytuł wymagany, limity długości)
- Szczegóły zadania wyświetlane w oknie dialog

## Technologie
- HTML
- CSS
- JavaScript (manipulacja DOM)
- localStorage

## Uruchomienie
Wystarczy otworzyć plik `index.html` w przeglądarce.
